Candle Light

Made in 48 hours for the Reddit Game Jam 5, the goal in this game is to find a hidden item inside a dark dungeon. As time passes, the light around the player becomes weaker. It also features line of sight mechanics throught which enemies are able to spot and chase the player character.

Made in Java using Slick2D.
