Caça Biscoitos

Caça Biscoitos is platformer game made for both the web and mobile devices.

Besides working on gameplay programming, I also made a remote controller for the desktop version of the game, where players could control the game using their mobile phones.

Made in Javascript using Pixi.js and Cocoon.js.

Gameplay programming: Lucas Tulio, Mauro de Tarso, Epa Neto, Endel Dreyer.

Game design: Epa Neto, Mauro de Tarso.

Graphics: Sérgio Tomo, Giordano Caldas.

Producer: Jéssica Blanco.
